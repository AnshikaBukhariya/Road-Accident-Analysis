# 🚧 Road Accident Analysis Dashboard (Excel)

## 📊 Project Overview
The **Road Accident Analysis Dashboard** is an interactive Microsoft Excel dashboard designed to analyze road accident data and identify patterns related to accident severity, vehicles, weather conditions, and road characteristics.

This project demonstrates how **Excel can be used for data analysis and visualization** to gain insights from accident datasets.

---

## 🎯 Project Objectives
- Analyze total road accident casualties.
- Understand accident severity (Fatal, Serious, Slight).
- Identify trends in accidents across months.
- Study accident distribution by vehicle type.
- Analyze the impact of road type, weather conditions, and speed limits.
- Compare accidents in urban and rural areas.
- Analyze accidents occurring during day and night.

---

## 📌 Dashboard Features

### 1️⃣ Total Casualties
Displays the total number of casualties recorded in the dataset.

### 2️⃣ Casualties by Severity
Breakdown of accidents into:
- Fatal
- Serious
- Slight

### 3️⃣ Casualties by Vehicle Type
Shows accident involvement by different vehicle types.

### 4️⃣ Monthly Trend Analysis
A line chart showing accident trends across different months.

### 5️⃣ Day vs Night Analysis
Shows accident distribution between daytime and nighttime.

### 6️⃣ Casualties by Location
Compares accidents in:
- Urban areas
- Rural areas

### 7️⃣ Casualties by Road Type
Accidents categorized by:
- Single carriageway
- Dual carriageway
- Roundabout
- One-way street
- Slip road

### 8️⃣ Casualties by Road Surface
Analyzes accidents based on road conditions:
- Dry
- Wet
- Snow / Ice

### 9️⃣ Interactive Filters
Users can filter data using:
- Speed limits
- Weather conditions
- Accident dates

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

---


## 📈 Key Insights
- Cars are involved in the highest number of accidents.
- Slight injuries occur more frequently than fatal or serious injuries.
- Single carriageway roads have the highest accident count.
- Wet road conditions increase accident risks.
- Accident frequency varies across months.
